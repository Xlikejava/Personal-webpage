## 4. 设计需求

### 4.1 视觉设计原则
**设计风格**：科技感、专业感、现代极简  
**核心关键词**：专业、可信、技术深度、优雅、响应迅速  

### 4.2 色彩系统
- **主色系**：深空蓝（#0f172a）、科技蓝（#2a6bc9）、夜间蓝（#1e293b）
- **辅助色**：琥珀色（#f59e0b）、青蓝色（#00c9c8）、成功绿（#10b981）
- **中性色**：纯白（#ffffff）、浅灰（#f8fafc）、中灰（#cbd5e1）、深灰（#475569）
- **语义色**：成功（#10b981）、警告（#f59e0b）、错误（#ef4444）、信息（#3b82f6）

### 4.3 排版系统
- **字体家族**：
  - 西文：Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', system-ui
  - 中文：PingFang SC, Hiragino Sans GB, Microsoft YaHei, sans-serif
- **字阶比例**：1.25倍递增（完美五度）
  - H1: 3.052rem / 4.5rem行高
  - H2: 2.441rem / 3.5rem行高  
  - H3: 1.953rem / 2.75rem行高
  - 正文: 1rem / 1.75rem行高
  - 小字: 0.8rem / 1.25rem行高

### 4.4 动效设计规范
1. **微交互**：
   - 按钮悬停：颜色加深 + Y轴位移2px + 阴影增强
   - 卡片悬停：Y轴位移-6px + 阴影扩散 + 边框高亮
   - 图标悬停：轻微旋转 + 颜色变化
   - 链接悬停：下划线展开动画

2. **页面级动画**：
   - 页面进入：渐入 + 轻微上浮（0.6s）
   - 滚动视差：背景层慢速，前景层正常
   - 元素入场：交错渐入（使用Intersection Observer）

3. **功能性动画**：
   - 加载状态：骨架屏 + 呼吸动画
   - 成功反馈：绿色打勾 + 缩放动画
   - 错误反馈：红色抖动 + 图标变化

### 4.5 质感表现
1. **玻璃态效果**：
   - 导航栏：半透明 + 背景模糊（backdrop-filter: blur(10px)）
   - 项目卡片：轻微半透明 + 内阴影 + 边框渐变

2. **深度层次**：
   - 使用多层级阴影（最多3层）
   - 层级高度：小（0-8px）、中（8-24px）、大（24-40px）

3. **细节处理**：
   - 圆角一致性：小（4px）、中（8px）、大（16px）、全圆（999px）
   - 边框：1px实线 + 半透明处理
   - 渐变：线性渐变（最多2色）避免过度设计

## 6. 技术需求

### 6.1 前端技术栈
- **核心**：HTML5、CSS3（Grid + Flexbox）、原生JavaScript（ES6+）
- **构建工具**：可选Vite或原生开发
- **第三方库**：
  - 动画：GSAP（仅复杂动画）、自实现CSS动画为主
  - 图标：FontAwesome或自设计SVG图标
  - 字体：Google Fonts（Inter字体）
  - 表单处理：Formspree或EmailJS
  - 代码高亮：Prism.js

### 6.2 性能要求
1. **加载性能**：
   - 首屏加载时间 < 2秒（Lighthouse评分 > 90）
   - 首次内容绘制（FCP） < 1.5秒
   - 最大内容绘制（LCP） < 2.5秒
   - 累积布局偏移（CLS） < 0.1

2. **资源优化**：
   - 图片：WebP格式 + 懒加载
   - CSS/JS：压缩 + 最小化
   - 字体：本地化加载 + subsetting
   - 第三方脚本：异步/延迟加载

### 6.3 响应式断点
- **移动端**：< 768px（垂直布局，简化交互）
- **平板端**：768px - 1199px（两列布局，适中元素）
- **桌面端**：≥ 1200px（完整布局，充分利用空间）
- **大桌面端**：≥ 1600px（优化阅读体验）

### 6.4 浏览器兼容性
- **主要支持**：Chrome >= 90、Firefox >= 88、Safari >= 14、Edge >= 90
- **降级支持**：IE不兼容，使用现代浏览器提示
- **特性检测**：对CSS Grid/Flexbox进行渐进增强
